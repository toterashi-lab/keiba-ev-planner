## 共通していた情報構造

1. 的中率・回収率・投資額・払戻額・収支の集計
2. 日付、競馬場、レース番号を軸にした直近実績一覧
3. 券種、買い目、点数、推奨金額の明示
4. 実績から個別レース詳細へ進む階層
5. 初見でも結果を比較できる強い数値階層

## このアプリで採用した内容

- 集計値から直近30レース明細へ進む二段構造
- 全シナリオを100円単位で揃えた比較
- 的中率だけでなく、投資・払戻・回収率・損益を同時表示
- 日付、競馬場、R、勝馬、オッズ、買い目、点数を一行で確認できる表
- 複数シナリオを切り替えるタブ

## 採用しなかった内容

- 会員登録や外部メッセージサービスへ誘導する広告バナー
- 独立検証できない高額払戻・利益表示
- 予測時点が不明なオッズを使った期待値実績
- 他社のブランド名、文章、画像、配色、固有の商品名

## 検証原則

掲載値は保存済みJRA公式結果と締切後単勝オッズからブラウザ内で再計算する。締切後オッズによる集計は市場ベンチマークであり、事前予測の成績とは表示しない。モデル実績は発走前オッズ、時系列分割、確率校正、データ漏洩検査がすべて合格するまで公開しない。
