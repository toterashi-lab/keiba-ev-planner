# 確率と期待値

市場確率は控除率を含む単純な逆数ではなく、全出走馬で正規化する。

`市場確率 = (1 / 単勝オッズ) / Σ(1 / 全馬の単勝オッズ)`

各エージェントは利用可能な特徴量だけを重み付きで合成し、レース内でsoftmax正規化する。欠損特徴量は0で補完せず、信頼度と品質ゲートへ反映する。

`適正オッズ = 1 / 校正済み勝率`

`100円あたり期待払戻 = 校正済み勝率 × 現在オッズ × 100`

`100円あたり期待利益 = 期待払戻 - 100`

複勝、馬連、ワイド、3連複、3連単は順位確率モデルから組合せ確率を導く。券種別の時系列校正が未完了なら`参考推定`と表示し、購入適格にはしない。
