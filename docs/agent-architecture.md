# エージェント設計

5エージェントは共通の発走前スナップショットを受け取り、使用特徴量、重み、最低EV、最低信頼度、券種、見送り条件を別々に持つ。

| エージェント | 主眼 | 主券種 |
| --- | --- | --- |
| 堅実派 セーフティ | 安定性、複勝圏、適性 | 複勝・ワイド・馬連 |
| 穴狙い スナイパー | 市場との乖離、不利補正、展開利 | 単勝・馬連・3連系 |
| 展開派 ペースメーカー | ペース、脚質、枠、隊列 | 馬連・ワイド・3連複 |
| 数理派 アナリスト | 校正確率、EV、誤差、標本数 | 検証済み券種 |
| 逆張り派 コントラリアン | 意見集中、過剰人気、分散 | 単勝・ワイド・3連複 |

各予想は`draft -> published -> locked -> settled`で遷移する。`published`以降は本体を更新・削除できない。結果確定後は精算とBrier Score、Log Loss、購入額、払戻額、収支を記録する。
